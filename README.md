# Advanced Chat App (Server)

Backend server for a real-time chat application using WebSocket.

The server handles message broadcasting between connected clients.

## Features

- Real-time communication
- WebSocket server
- Multi-client support

## Tech Stack

- Node.js
- WebSocket
- JavaScript

## Installation

git clone https://github.com/rajeshr-25/advancedChatAppServer.git

cd advancedChatAppServer

npm install

node server.js
